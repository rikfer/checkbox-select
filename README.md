# React + TypeScript + Ricardo Ferrari

Multiple select checkboxes for the React exam.

## Installation
Clone the repository:

```bash
$ git clone https://github.com/rikfer/meli-challenge.git
```
Install with npm...
```js
# npm
npm install

# pnpm
pnpm install
```

Run locally
```js
# pnpm
pnpm dev 

# npm
npm run dev
```
